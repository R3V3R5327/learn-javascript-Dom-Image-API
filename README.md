# 🖼️ Image Gallery App (API Based)

A simple JavaScript project that fetches images from an API and displays them dynamically on a webpage.

---

## 🚀 Features

* Fetch images from an external API
* Display images dynamically using JavaScript
* Show author name for each image
* Responsive and clean UI
* Beginner-friendly project

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Fetch API

---

## 📡 API Used

This project uses the following API:

```
https://picsum.photos/v2/list
```

---

## 📂 Project Structure

```
├── index.html
├── style.css
├── scripts.js
└── README.md
```

---

## ⚙️ How It Works

1. JavaScript uses `fetch()` to call the API
2. API returns a list of images (JSON format)
3. Each image object contains:

   * Image URL
   * Author name
4. JavaScript dynamically creates:

   * `<img>` for image
   * `<p>` for author
5. All elements are appended to the DOM
